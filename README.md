# Smarteye-Query-Bot-V1

SmartEye Query Bot is an internal analytics assistant that allows users
to query attendance and labour data using natural language.

## Features
- Employee attendance by employee code
- Double duty & triple duty detection
- Overtime and outsider labour analysis
- Date-wise and month-wise attendance
- Mill-wise filtering (SHJM, SHMM, etc.)

## Tech Stack
- Python
- Streamlit
- SQL Server
- FastAPI (if backend exists)

## How to Run
1. Install dependencies
   pip install -r requirements.txt
2. Set environment variables
3. Run:
   streamlit run app.py
